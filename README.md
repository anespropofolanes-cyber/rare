# 兒科罕見症候群．麻醉查詢 — 使用與維護說明

一份離線可用的單一網頁，整合 50 個兒科罕見症候群的麻醉參考資料，並附臨床指南與實務工具包。本文件供使用者與維護者參考。

---

## 一、這是什麼

- **檔名**：`兒科罕見症候群麻醉查詢.html`（單一檔案，約 8 MB）
- **用途**：臨床教學與圍術期快速參考。以「麻醉須知」為核心，輔以雙軸分類、原始整理圖卡、編碼對照與臨床指南。
- **特性**：純前端、**完全離線可用**。所有資料（含圖片）都內嵌在這一個檔案裡，不需要網路、不需要安裝、不依賴外部資料夾。

---

## 二、開啟方式

1. 用任何現代瀏覽器（Chrome、Edge、Safari、Firefox）直接開啟 `.html` 檔即可。
2. 手機可用「檔案」App 或瀏覽器開啟；也可加到主畫面當書籤。
3. 第一次開啟因內含圖片會稍久，之後順暢。
4. **搬移／分享時請保持單一檔案完整**，不要拆檔；複製整個 `.html` 即可在其他裝置使用。

---

## 三、主要功能

### 1. 雙軸分類查詢
- 上方「分類方式」可切換 **依麻醉風險 / 依病因** 兩種分類軸。
- 下方下拉選單選擇分類即篩選；選「全部」等於清除。
- **麻醉風險軸（10 類，可多標籤）**：困難呼吸道、術後呼吸併發症、惡性高熱／藥物注意、心血管受累、出血傾向、血栓風險、代謝／低血糖／禁食不耐受、神經系統（含癲癇）、多系統／複雜、一般處置（風險較低）。
- **病因軸（7 類）**：染色體數目異常、染色體缺失／微缺失、基因印記異常、單基因疾病、體細胞鑲嵌／非遺傳、序列／關聯、其他。

### 2. 搜尋
- 上方搜尋框可搜：症候群中／英文名、別名、基因／位點、ORPHA／ICD-11 編碼、臨床特徵、麻醉須知。

### 3. 症候群詳情
點任一卡片開啟，內容包含：
- **原始整理圖卡**（點圖可全螢幕放大）
- **麻醉人員麻醉須知**（核心，暖色面板）
- 重點關聯、臨床特徵
- 病因與編碼（基因／位點、遺傳方式、發病年齡、最常見心臟缺損、ORPHA、ICD-11）
- 診斷、預後
- **風險徽章跳轉**：標有「›」的風險徽章可點，直接跳到「臨床指南」對應分頁（困難呼吸道、惡性高熱、代謝、出血／血栓→凝血功能）。

### 4. 臨床指南（頂部「臨床指南」按鈕）
點開後分兩大區：

**風險架構（7 分頁）**
風險總則、惡性高熱、困難呼吸道、代謝疾病、凝血功能、緊急評估、診斷流程。

**實務工具包（4 分頁）**
個人執業者、麻醉護理師、快速參考卡（DMD／惡性高熱）、資源庫。

- 各分頁內的**檢核清單可直接點選打勾**（打勾後文字劃線）。
- 勾選狀態**僅當次有效，重新整理即清空**——這是刻意設計，避免把上一個病人的勾選帶到下一個。

### 5. 管理員（頂部「管理員」按鈕）
- 需密碼進入（密碼請洽本檔建立者；修改方式見下方第五節）。
- 可**新增**自訂症候群、**管理／刪除**已新增項目、**匯出／匯入** JSON 備份。
- 新增的資料儲存在該瀏覽器本機（localStorage），**不會寫回 HTML 檔本身**，也不會跟著檔案複製到別台裝置——要轉移請用「匯出 JSON」再到另一台「匯入」。

## 四、資料與分類邏輯

- 收錄 **50 個症候群**；每個圖卡與資料一一對應。
- 「麻醉須知」依一般圍術期考量整理；**藥物名稱一律以英文呈現**（如 succinylcholine、propofol、dantrolene）。
- ORPHA／ICD-11 編碼為對照之用。
- 麻醉風險為多標籤（一個症候群可同時屬於多類），卡片上的色標為其「主風險」。

## 五、修改細節（給維護者）

> ⚠️ **修改前務必先備份**整個 `.html` 檔（複製一份另存）。本檔約 8 MB 且內含 base64 圖片，建議用能處理大檔的純文字編輯器（VS Code、Sublime、Notepad++）開啟，**不要用 Word**。
> 以下修改是直接編輯這個 HTML 檔；較大的改動（新增分頁、改版面、換圖）建議把檔案交回給原協作的 AI 助理處理，較不易出錯。

### 改管理員密碼
1. 在檔案中搜尋：`ADMIN_PASS`
2. 會找到類似 `const ADMIN_PASS="******";`
3. 把引號內的數字改成新密碼，存檔即可。
4. （登入畫面的「靈感先生」提醒文字，可搜尋 `gatehint` 附近的文字修改。）

### 改配色
1. 搜尋 `:root{` ，裡面是一組色彩變數，改十六進位色碼即可全站套用：
   - `--bg`：頁面背景（目前 `#F4F3E4`，淡黃灰）
   - `--surface`：卡片底色（白）
   - `--ink`：主要文字色
   - `--teal` / `--teal-d`：主重點色（橄欖綠 `#8A8C40`／深橄欖）
   - `--amber-bg`：麻醉須知面板底色（`#FFF9B3` 淡黃）
2. 標題列漸層另外設定：搜尋 `header.top{background:linear-gradient` ，改兩個色碼。

### 新增／刪除症候群
- **建議用「管理員」面板**操作（不需動原始碼），再用「匯出 JSON」保存。
- 若一定要改原始碼：搜尋 `const DB = {"items"` 可找到資料陣列，但因內含大量 base64 圖片資料、且格式嚴謹，**手動編輯極易出錯，不建議**。

### 改指南文字
- 直接在檔案中搜尋畫面上看到的那段中文字，找到後修改即可。改完存檔、重新整理瀏覽器確認。

### 登入畫面的圖片
- 搜尋 `gateimg` 可找到該圖（base64 內嵌）。更換需替換 base64 字串，建議交回 AI 助理處理。
