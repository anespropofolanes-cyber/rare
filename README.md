# 兒科罕見症候群．麻醉查詢 — 使用與維護說明

一份離線可用的單一網頁，整合 50 個兒科罕見症候群的麻醉參考資料、臨床指南與實務工具包。本文件供使用者與維護者參考。

---

## 一、這是什麼

- **檔名**：`兒科罕見症候群麻醉查詢.html`（單一檔案）
- **用途**：臨床教學與圍術期快速參考。以「麻醉須知」為核心，輔以雙軸分類、原始整理圖卡、編碼對照與臨床指南。
- **特性**：純前端、**完全離線可用**。所有資料（含圖片）都內嵌在這一個檔案裡，不需網路、不需安裝、不依賴外部資料夾。

---

## 二、開啟方式

1. 用任何現代瀏覽器（Chrome、Edge、Safari、Firefox）直接開啟 `.html` 檔。
2. 手機可用瀏覽器開啟，並可「加入主畫面」，會顯示專屬 App 圖示、像 App 一樣全螢幕開啟。
3. 第一次開啟因內含圖片會稍久，之後順暢。
4. **搬移／分享時請保持單一檔案完整**，複製整個 `.html` 即可在其他裝置使用。

---

## 三、主要功能

### 1. 雙軸分類查詢
- 上方「分類方式」可切換 **依麻醉風險 / 依病因**。
- 下方下拉選單選擇分類即篩選；選「全部」等於清除。
- **麻醉風險軸（10 類，可多標籤）**：困難呼吸道、術後呼吸併發症、惡性高熱／藥物注意、心血管受累、出血傾向、血栓風險、代謝／低血糖／禁食不耐受、神經系統（含癲癇）、多系統／複雜、一般處置（風險較低）。
- **病因軸（7 類）**：染色體數目異常、染色體缺失／微缺失、基因印記異常、單基因疾病、體細胞鑲嵌／非遺傳、序列／關聯、其他。

### 2. 搜尋
- 上方搜尋框可搜：中／英文名、別名、基因／位點、ORPHA／ICD-11 編碼、臨床特徵、麻醉須知。

### 3. 症候群詳情
點任一卡片開啟，包含：原始整理圖卡（點圖可放大）、麻醉人員麻醉須知（核心）、重點關聯、臨床特徵、病因與編碼（基因／位點、遺傳方式、發病年齡、最常見心臟缺損、ORPHA、ICD-11）、診斷、預後。
- **風險徽章跳轉**：標有「›」的風險徽章可點，直接跳到「臨床指南」對應分頁（困難呼吸道、惡性高熱、代謝、出血／血栓→凝血功能）。

### 4. 臨床指南（頂部「臨床指南」按鈕）
點開後分兩大區：

**風險架構（7 分頁）**：風險總則、惡性高熱、困難呼吸道、代謝疾病、凝血功能、緊急評估、診斷流程。

**實務工具包（4 分頁）**：個人執業者、麻醉護理師、快速參考卡（DMD／惡性高熱）、資源庫。

- 各分頁內的**檢核清單可直接點選打勾**（打勾後文字劃線）。
- 勾選狀態**僅當次有效，重新整理即清空**——刻意設計，避免把上一個病人的勾選帶到下一個。

### 5. 管理員（頂部「管理員」按鈕）
- 需密碼進入（密碼請洽本檔建立者；修改方式見第五節）。
- 可**新增**自訂症候群、**管理／刪除**已新增項目、**匯出／匯入** JSON 備份。
- 新增資料儲存在該瀏覽器本機，**不會寫回 HTML 檔**，也不會跟著檔案複製到別台裝置——要轉移請用「匯出／匯入」。

## 四、資料與分類邏輯

- 收錄 **50 個症候群**，每個圖卡與資料一一對應。
- 「麻醉須知」依一般圍術期考量整理；**藥物名稱一律以英文呈現**（如 succinylcholine、propofol、dantrolene）。
- ORPHA／ICD-11 編碼為對照之用。
- 麻醉風險為多標籤；卡片上的色標為其「主風險」。

## 五、修改細節（給維護者）

> ⚠️ **修改前務必先備份**整個 `.html` 檔（複製一份另存）。檔案較大且內含內嵌圖片，建議用純文字編輯器（VS Code、Sublime、Notepad++）開啟，**不要用 Word**。
> 以下是直接編輯這個 HTML 檔；較大的改動（新增分頁、改版面、換圖、改功能）建議把檔案交回給原協作的 AI 助理處理，較不易出錯。

### 改管理員密碼
1. 搜尋：`ADMIN_PASS`
2. 找到 `const ADMIN_PASS="******";`，把引號內的值改成新密碼，存檔。
3. 登入畫面的「靈感先生」提醒文字，可搜尋 `gatehint` 附近修改。

### 改配色
搜尋 `:root{`，改十六進位色碼即可全站套用（目前現值）：
- `--bg`：頁面背景（`#F4F3E4`，淡米黃）
- `--surface`：卡片底色（白）
- `--ink`：主要文字色
- `--teal` / `--teal-d`：主重點色（橄欖綠 `#8A8C40`／深橄欖，用於按鈕、選取、編號徽章）
- `--amber-bg`：麻醉須知面板底色（`#E8EBC6`，淡橄欖綠）

標題列漸層另外設定：搜尋 `header.top{background:linear-gradient`，目前為深綠 `#3C5429 → #304521`，改兩個色碼即可。

### 新增／刪除症候群
- **建議用「管理員」面板**操作（不需動原始碼），再用「匯出 JSON」保存。
- 若要改原始碼：搜尋 `const DB = {"items"` 可找到資料陣列，但因含大量圖片資料、格式嚴謹，**手動編輯極易出錯，不建議**。

### 改指南文字
- 直接在檔案中搜尋畫面上看到的那段中文字，找到後修改、存檔、重新整理確認。

### 換登入畫面圖片或 App 圖示
- 登入圖：搜尋 `gateimg`；App 圖示：搜尋 `apple-touch-icon`。皆為 base64 內嵌，更換建議交回 AI 助理處理。
